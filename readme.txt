Requerimientos generales:
Usted ha sido contratado para programar la siguiente maqueta web que permite registrar un cliente y la compra de un producto. Premisa: en el proceso de registro se registra un cliente y un producto a la vez; es decir, un cliente puede comprar un unico producto. Premisa: se asume que el usuario siempre completa los inputs sin dejar ninguno por contestar, por lo que no es requerido validar el formulario.

Programar ademas que una vez completada la variable con esa informacion se agregue un elemento de lista a la "Lista de Clientes"

2.- Programar que con los datos del formulario Registrar "Venta de Producto" se cree un arreglo con esos datos.

3.- Programar ademas que con los datos del arreglo del requerimiento numero 2, se cargue una nueva fila de la "Tabla de Productos Vendidos"

4.- Programar ademas que tanto los elementos de la lista como las filas de la tabla tengan el evento doble click, este debe primero solicitar confirmacion a traves de un confirm si desea eliminar el elemento.

5.- Una vez cargado los datos la variable auxiliar, el arreglo auxiliar y los inputs de los formularios usados deben reestablecer su valor.
