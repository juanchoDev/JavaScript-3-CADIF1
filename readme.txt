Requerimientos generales:
Usted ha sido seleccionado para completar las validaciones front end del formulario de registro de nuestra empresa. Para ello se le suministrara el formulario y debera escribir los scripts necesarios para alcanzar cumplir con los requerimientos que se listan a continuacion.

Requerimientos específicos:
1.- Validar con el uso de una expresion regular que el campo de nombre y apellido solo admita carateres alfanumericos, sin caracteres especiales. Advertir del error con un alert.

2.- Validar con el uso de una expresion regular que el campo de correo sea escrito un formato de correo valido. Advertir del error con un alert.

3.- Validar con el uso de ua expresion regular que la contraseña solo admita caracteres alfanumericos y que como minimo tenga 6 caracteres y como maximo 10 caracteres. Advertir del error con un alert.

4.- Validar que sea seleccionada una ciudad del select de ciudad indicado. Advertir del error con un alert. Validar que el checkbox de "terminos y condiciones" sea seleccionada para poder registrarse.

5.- Validar ademas que ninguno de los 4 primeros inputs del formulario esten vacios. Advertir del error con un alert.
